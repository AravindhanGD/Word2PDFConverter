# 📄 Word to PDF Converter

A simple web application that converts Word documents (.docx) to PDF files while preserving formatting, tables, and basic styles.

## ✨ Features
- Upload any `.docx` file
- Converts to PDF with the same filename
- Preserves formatting, tables, and text styles
- Clean, mobile-friendly interface
- Works on Windows, Mac, and Linux
- Completely free and open source

## 🚀 How to Use
1. Upload your Word document
2. Click "Convert to PDF"
3. Your PDF will automatically download

## 🛠️ Technology Stack
- **Backend**: Flask (Python)
- **Word Processing**: python-docx
- **PDF Generation**: ReportLab
- **Frontend**: HTML, CSS, JavaScript
